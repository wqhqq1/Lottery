##Build
####If you want to build the code by yourself, you should install the depends in the list:
1. build environment
   - Python 3.7 or higher
2. python wheels
   - pyperclip
   - xlrd
   - xlwt
   - pyinstaller
3. run this codes in terminal
```
git clone
cd ./Lottery
python3 ./Main.py
```
